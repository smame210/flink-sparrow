# flink-sparrow

## 介绍
这是一个旨为快速构建flink任务的项目。

## 项目结构
```
flink-sparrow
├── README.md              # 项目说明
├── sparrow-api            # api接口
├── sparrow-common         # 公共模块
├── sparrow-core           # 核心模块，包含flink的启动入口
├── sparrow-connector      # 非官方的flink连接器
├── sparrow-plugin         # 自定义插件
├── sparrow-support        # 一些工具，例如脚本执行器 
```
